# **App Name**: ExpenseWise

## Core Features:

- Expense Entry: Allow users to input their daily expenses with categorization (fuel, food, travel, etc.).
- Table View: Display a list of all expenses in a table format, sorted by date, category, and amount.
- Analytics Dashboard: Provide visual analytics and charts to track spending patterns weekly and monthly.
- AI Budget Tool: Based on spending patterns, give users recommendations for savings.

## Style Guidelines:

- Primary color: Teal (#008080) to give the feeling of fiscal calm.
- Background color: Light gray (#E0E0E0), very slightly desaturated to be neutral.
- Accent color: Green (#00A300) to complement teal and emphasize savings. 
- Body and headline font: 'Inter', sans-serif.
- Use minimalistic icons for expense categories to maintain a clean look.
- Table layout for expense lists, card-based layout for dashboard elements.
- Smooth transitions between views and subtle animations on chart updates.
